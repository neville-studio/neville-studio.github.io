This Page shows the Open Java Development Kit for Neville Builds, which are not modified anyway and for provided as is. We do not have TCK to test, we just to see if it can be successfully built and can run some simple Java code.

Why we build this OpenJDK? Because I want to running the OpenJDK in some Old Operating System. Especially Windows. We will keep capability that Open JDK Build docs say, And we will do not upgrade the any build tools for use.

Supported Platforms:

  • Windows: Windows XP or Windows Server 2003(Open JDK 8, but only part of the API do work)
  • Windows: Windows Vista or Windows Server 2008(Open JDK 8, all the api work properly.)
  • Windows: >=Windows 7 or Windows Server 2008 R2(>=Open JDK 8, all the api work properly. Be sure that the operating system is fully upgraded)
  • Linux: GLIBC >= 2.12(Open JDK 8 or greater, some features require fontConfig and x11 related packages)
  • Linux: 2.9 <= GLIBC < 2.12 (Open JDK 8 supports, some features require fontConfig and x11 related packages)
  • Other: Do not support
  • Architecture: Open JDK 8: x86_64, x86; >=OpenJDK 11,Only x64

Warning: do not use this OpenJDK Builds for production! You can go to Eclipse Adoptium's website to get a official stable release.
