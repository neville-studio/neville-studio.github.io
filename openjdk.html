This Page shows the Open Java Development Kit for Neville Builds, which are not modified anyway and for provided as is. We do not have TCK to test, we just to see if it can be successfully built and can run some simple Java code. All the system requirement is built for

Why we build this OpenJDK? Because I want to running the OpenJDK in some Old Operating System. Especially Windows. We will keep capability that Open JDK Build docs say, And we will do not upgrade the any build tools for use.
